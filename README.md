# 货运刷题

铁路货运组织 I 离线刷题网页。

## 功能

- 单选题按序号分 3 组。
- 判断题按序号分 3 组。
- 多选题 1 组。
- 错题本自动保存到浏览器本地存储。
- 答题过程中不显示答案，可用上一题/下一题切换并修改答案。
- 全组答完后点击交卷，统一显示本组答题情况、你的答案、正确答案和选项颜色标记。
- 支持电脑和手机浏览器访问。

## 选项颜色

- 绿色：正确选项。
- 黄色：少选的正确选项。
- 红色：选错的选项。

## 本地使用

双击 `打开刷题软件.bat`，或直接打开 `index.html`。

## 上传到 GitHub

把本文件夹内的文件上传到 GitHub 仓库即可。开启 GitHub Pages 后，访问 Pages 地址就能在手机端刷题。错题本会保存在当前浏览器中，更换浏览器或设备后不会自动同步。

## 重新生成题库

题库数据已生成在 `questions.js`。如果以后替换 PDF 题库，可以运行：

```bash
python build_questions.py
```
